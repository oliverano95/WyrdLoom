# Wyrd Loom for Pebble ⌚

Wyrd Loom is a highly optimized, feature-rich, and fully customizable analog watchface for the Pebble smartwatch ecosystem.

## ✨ Features
* **Buttery Smooth Animations:** The minute hand and virtual camera pan smoothly to the current time every minute.
* **Dynamic Complication Pill:** Choose from 14 native Pebble metrics (Steps, Heart Rate, Compass, Battery, etc.) to display in a mechanically rotating center pill.
* **Extreme Battery Optimization:** Uses off-screen culling, trigonometric caching, and smart hardware polling to ensure maximum battery life.
* **Theme Import/Export:** Easily share your custom colorways with a Base64 theme code.
